Enregistrement et connexion

  • Pour pouvoir communiquer sur DevChat, vous devez obligatoirement vous connecter. Vous pouvez néanmoins consulter les messages en tant qu'utilisateur non connecté, mais la fonction d'envoi de message n'est pas accessible. Pour toute action, une connexion sera demandée.

  • Chaque utilisateur doit posséder un identifiant unique. Vous ne pouvez pas choisir un pseudonyme déjà utilisé (le cas échéant, un message apparaîtra pour vous signaler que le nom est déjà pris). Vous pourrez éditer votre nom d'utilisateur autant de fois que vous le souhaitez.

  • Lorsque vous ouvrez une session, un message s'affiche sur le chat pour indiquer aux autres utilisateurs que vous avez rejoint le canal. Un onglet affiche en temps réel le nom de tous les utilisateurs connectés. Votre session se ferme dès que vous quittez le chat.

Utiliser DevChat

  • DevChat est conçu pour être intuitif et simple d'utilisation. Vous pouvez écrire autant de messages que vous voulez ; il vous suffit pour cela de saisir votre texte dans le champ en bas de l'écran, puis de l'envoyer en appuyant sur la touche « Entrée » de votre ordinateur ou en cliquant sur le bouton « envoyer ». Il est bien entendu impossible d'envoyer un message vide. Pour les sauts de lignes, appuyez sur Shift + Entrée.

  • Lorsque vous envoyez un message, il est immédiatement stocké sur notre serveur et devient visible de tous les utilisateurs. La date et l'heure de publication sont indiqués sur l'en-tête de chaque message. Vous pouvez à tout moment éditer votre message à l'aide du bouton dédié (les modifications seront visibles instantanément par tous les utilisateurs).

  • Tous les messages que vous envoyez sont sauvegardés dans notre base de données. Un bouton « Messages précédents » vous permet de remonter le fil de la conversation et revoir les anciens messages. Vous pouvez également supprimer vos messages ; ils seront alors effacés de notre base de données et ne seront plus visibles des autres utilisateurs.

  • Pour fermer votre session en cours, utilisez le bouton « Déconnexion ».

  • Pour commencer à utiliser DevChat, cliquez sur le bouton « Accéder au chat » dans l'en-tête de la page.
